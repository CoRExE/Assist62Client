## How to run

### Frontend

- Front Render

```bash
pnpm dev
```

### Backend

- Backend Render

```bash
pnpm tauri:d
```

- Backend Build

```bash
pnpm tauri:b
```

- Backend Clean

```bash
pnpm tauri:clean
```
